System Overview

CONSULT-IQ​ uses a group-chat orchestration framework where a central Orchestrator analyzes the user's request and routes it to the most relevant medical specialist agents. Each specialist agent represents a different domain of expertise, such as Oncology, Pediatrics, Urology, and Neurology.  

Flow:

  1. The User submits a medical question.
  2. The Orchestrator determines which specialist agents are relevant.
  3. Selected agents collaborate and exchange responses.
  4. Their reasoning and discussion are visible to the user.
  5. The system returns a more comprehensive, multi-perspective answer.

Knowledge

Each agent uses a dual-layer knowledge system:
 
1. Global Knowledge:
 Shared, specialty-specific information from curated medical knowledge bases (via Microsoft Foundry IQ). Ensures accurate and consistent expertise.
 
2. Local Knowledge:
 User-specific notes and documents (PDFs, text) that customize the agent’s behavior. Private and only visible to that user.
Agents combine both layers to deliver answers that are medically reliable and personalized.
